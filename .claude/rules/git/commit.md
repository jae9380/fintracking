# Commit Convention

## Format

{type}: {title}

{body - optional}

## Types

- feat: new feature
- fix: bug fix
- refactor: code improvement
- test: test changes
- docs: docs update
- chore: build/config

## Examples

feat: add JWT refresh API
• validate refresh token
• auto renew expired token

```
fix: missing duplicate check on account create
```

## Forbidden

- No period in title
- No vague messages
- No multiple features in one commit
